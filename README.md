# Paradise Nursery 🌿

Paradise Nursery is a React-based plant shopping web application where users can browse plants and add them to a shopping cart.

## Features
- Browse plants by categories
- Add plants to cart
- Increase/decrease quantity
- Remove items from cart
- Checkout functionality

## Technologies Used
- React.js
- Redux Toolkit
- CSS

## Installation
1. Clone the repository
2. Run npm install
3. Run npm start

## Usage
Users can explore different plants, add them to their cart, and manage their shopping experience easily.
